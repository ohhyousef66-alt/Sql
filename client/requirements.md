## Packages
recharts | Data visualization for vulnerability graphs
framer-motion | Smooth animations for dashboard elements and page transitions
date-fns | Formatting timestamps in logs and reports
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes safely

## Notes
- Dark mode is default and mandatory for the cybersecurity aesthetic.
- Terminal logs should use a monospace font.
- Vulnerability severities map to specific colors (Critical=Red, High=Orange, Medium=Yellow, Low=Blue, Info=Gray).
